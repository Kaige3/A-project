# 模板1：初期思路雏形·AI决策纠偏（最常用）
适用：只有模糊想法，让AI帮你梳理、挑错、定方向
```
我现在有一个初期思路/方案雏形：【简要描述你的想法、目标、要解决的问题】
当前约束条件：【时间/成本/技术/人员/风险等限制，没有就写无】
请帮我：
1. 梳理核心决策点，指出思路漏洞/不合理之处
2. 给出优化方向和修正建议
3. 基于此给出清晰的决策结论（可行/调整/放弃）
4. 输出可落地的核心执行框架
```

# 模板2：多方案对比·AI优选决策
适用：有2-3个方案，让AI帮你选最优、排优先级
```
我需要在以下方案中做决策：
方案A：【描述方案A】
方案B：【描述方案B】
方案C：【描述方案C】
决策维度：【可行性/成本/效率/风险/收益/落地难度】
约束条件：【核心限制】
请用SWOT+RICE模型分析各方案优劣，给出明确最优方案，并说明理由、潜在风险及应对措施。
```

# 模板3：决策定案后·一键输出精简PRD+SOP
适用：已经确定方案，直接让AI生成标准文档
```
基于以上最终决策方案：【复述最终确定的方案核心】
请输出两份内容：
一、精简版PRD（产品需求）
1. 业务目标 2. 核心需求 3. 实现逻辑 4. 验收标准
二、标准化SOP（执行流程）
1. 适用范围 2. 执行步骤 3. 责任人/节点 4. 异常处理 5. 检查标准
要求结构清晰、可直接落地使用。
```

# 模板4：快速头脑风暴+风险兜底决策
适用：想发散思路，同时避免踩坑
```
针对【你的核心问题/目标】，先做头脑风暴，给出3种不同的实现思路；
再对所有思路做风险评估，标注高风险点和规避方案；
最后给出最稳妥的决策建议，以及下一步可执行的1-3个动作。
```
